# Real-Time Anti-Doping Monitoring System

This project is a real-time anti-doping system for athletes.

Features:
- Monitor athlete health data in real time
- Detect abnormal values using AI
- Send alert if doping suspected
- Store data in database
- Show live dashboard

Tech stack:
Frontend: HTML, CSS, JS
Backend:  Python
Database: MySQL
AI: Python
Realtime: Firebase / Socket
